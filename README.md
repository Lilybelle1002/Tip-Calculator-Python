# Python Tip Calculator

## Project Overview
This project is a simple Python program that calculates the tip for a restaurant bill and splits the total between multiple people. The program asks the user for the bill amount, the tip percentage, and the number of people sharing the bill. It then calculates the tip, the total bill including the tip, and how much each person should pay.

## Purpose
The purpose of this project is to practice basic Python programming concepts such as user input, variables, mathematical calculations, and displaying results. This project was also used to demonstrate how GitHub can store and manage code projects.

## How the Program Works
The program performs the following steps:

1. Asks the user to enter the bill amount.
2. Asks the user to enter the desired tip percentage.
3. Asks how many people are splitting the bill.
4. Calculates the tip amount.
5. Calculates the total bill, including the tip.
6. Divides the total bill by the number of people.
7. Displays the tip amount, total bill, and amount each person should pay.
